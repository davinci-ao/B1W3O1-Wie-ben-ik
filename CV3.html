Wat doe ik na de lessen

Ik ben bereid om na de lessen door te werken aan projecten zo nodig. Ik vind de opleiding harstikke leuk dus werk ik ook graag thuis aan een project dat ik zelf heb bedacht wat niet hoeft voor school, maar natuurlijk doe ik ook gewoon thuis werken voor schoolprojecten die afmoeten. Dit doe ik zodat ik meer oppak en dat ik qua kennis al voorloop op de lessen. Zoals op dit project heb ik geleerd hoe je buttons moet customizen met css, dit heb ik allemaal thuis geleerd. Ook heb ik thuis een wat beter focus thuis omdat ik dan alleen zit en er zijn dan geen aflijdingen.
