Ontwikkelpunten

Als ontwikkelpunt zou ik graag mijn eigen referentiekader willen ontstijgen om mijn kennis breder te maken, zodat ik meerdere manieren kan bedenken om een probleem te omzeilen en tot een oplossing te komen. Daarnaast, gezien ik nog beginnende ben in de opleiding, wil ik graag de technieken goed aanleren, en de theorie achter de code die ik schrijf, begrijpen. Java heeft bijvoorbeeld “public static void main(String[] args)”. Iedereen kan die code kopiëren en gebruiken, maar als je pas echt weet wat iedere woord betekent, weet je pas echt wat je aan het doen bent. Hierdoor wordt het weer makkelijker om oplossingen te bedenken op vraagstukken. Ik zie het graag als een vorm van puzzelen.
